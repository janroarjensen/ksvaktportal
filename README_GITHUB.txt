GitHub Pages – steg for steg

1. Opprett et repository på GitHub.
2. Last opp innholdet i denne mappen til repository-roten.
3. Åpne index.html og lim inn verdiene i window.FIREBASE_CONFIG.
4. Gå til Settings -> Pages.
5. Velg Deploy from a branch.
6. Velg main og / (root).
7. Etter 1–3 minutter er siden live på GitHub Pages.

Merk: GitHub Pages hoster den statiske siden, mens Firebase Realtime Database brukes for live synk.
